## 1. Old Format to New Format Question Migration

**Summary:** The old format questions directory (`questions/validated/`) contained 19 files with ~280 questions that needed migration to the new structured format (`history-data/{epoch}/{chapter}/`). The old format lacked proper YAML frontmatter, standardized structure, and validation. Created a reusable migration instruction file and executed parallel subagents to migrate 93 questions across 6 chapters, with special corrections for curriculum-aligned difficulty classification. Goal was to consolidate all questions into the new format and eliminate the old directory.

**Type:** `backend`
**Scope:** `large`
**Tokens:** Input: 98,400 | Output: 73,200 | Total: 171,600
**Time:** Session: 90 min | Model: 45 min

### What Changed
- ✅ Created comprehensive migration instruction file (`ai-instructions/question-migration-instruction.md`) for reusable subagent workflow
- ✅ Migrated 93 questions from old format to new structured format across 6 chapters (Ekspansja, Rozbicie dzielnicowe I, Słowianie, plus cleanup of 3 already-migrated chapters)
- ✅ Corrected difficulty classifications for 10 questions (Rozbicie dzielnicowe I EASY→MEDIUM) based on curriculum alignment analysis
- ✅ Validated all migrated questions with MCP polish-history tools, polish-grammar-checker, and difficulty-reviewer skill
- ✅ Updated state files (questions-tracker.json, state.json) with migration details and final counts
- ✅ Deleted 19 old format files, consolidating all questions into new format structure
- ✅ Applied 5 grammar corrections during migration (spelling, punctuation, case agreement)

### Technical Approach
- **Parallel subagent execution:** Launched up to 3 subagents simultaneously to process different chapters in parallel, maximizing efficiency
- **Curriculum-based difficulty validation:** Verified all questions against Polish curriculum standards (szkola_podstawowa_zpe.md for EASY, liceum_technikum_zpe.md for MEDIUM/HARD)
- **Six-phase workflow per chapter:** (1) Preparation/analysis, (2) Historical validation with MCP tools, (3) Format conversion, (4) Quality validation (grammar + difficulty), (5) State management, (6) Cleanup/commit
- **Special handling:** Identified and corrected curriculum misalignment where EASY-level topics (Bolesław Krzywousty, pradzieje) actually belong in high school curriculum
- **Quality gates:** Each migration required 0 grammar errors and correct difficulty classification before completion
